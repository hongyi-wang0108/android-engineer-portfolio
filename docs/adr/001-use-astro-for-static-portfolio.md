# ADR 001: Use Astro for the portfolio site

## Status

Accepted

## Context

The project is a content-heavy static portfolio with a small amount of interaction. It needs fast loading, simple deployment, and low maintenance cost.

## Decision

Use Astro as the application framework, with TypeScript and Tailwind CSS.

## Consequences

- Static generation stays simple
- Build and deploy cost stay low
- Content pages are easy to maintain
- Future interactive islands can be added selectively if needed
